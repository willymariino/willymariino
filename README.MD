### Featured Projects
- [TechZone](https://github.com/willymariino/techzone-spec-frontend) – Full stack e-commerce web application built with React, Context API, Node.js, Express, and MySQL. Features product search with debouncing, filtering, side-by-side comparison, persistent wishlist and shopping cart, dynamic REST API integration, and responsive UI architecture.

- [Urbn Way](https://github.com/willymariino/Urb4n-Way) – Full stack e-commerce web application built with React, Node.js, Express, and MySQL. Features product search and filtering, wishlist and shopping cart management, user registration, checkout flow, email integration, and dynamic REST API communication. Developed as a collaborative team project.

- [BooRoad](https://github.com/willymariino/booroad) – React-based travel management application designed to help trip leaders organize journeys and manage travelers. Built with React and Vite, it features trip and guest management, capacity validation, traveler contact tools, and dynamic data handling through a responsive interface. Developed as a collaborative team project.

- [Lodi Weather App](https://github.com/willymariino/weather-web-app) – React weather dashboard that displays current weather conditions for Lodi, Italy using the Open-Meteo API. Shows temperature, humidity, wind speed and sky conditions in a simple card-based UI, with manual refresh and an animated cloud background.
